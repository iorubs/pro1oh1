Docker:

How Docker made it all possible:

A lot of tools were used to develope pro1oh1, but the one that made it all possible was Docker.

Without it, creating sandboxed environments, running user applications and collecting stats would have been too complex.

One of the end goals is to one day fully dockerize the application. So I can share it with the world.

What's docker?

Docker is an open-source project that automates the deployment of applications inside software containers, by providing an additional layer of abstraction and automation of operating-system-level virtualization on Linux.

Docker Hub:

The Docker Hub lets developers store and distribute their images with ease. Over 1.4 billion images pulls have already taken place.

Many of the images available, come straight from official sources like Google and Oracle, with pre-baked tools.
