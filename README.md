# Pro1oh1 ODE.
######(online development environment)

## Instructions
1- Install docker and docker-compose.  
2- `git clone https://github.com/vasconr2/pro1oh1.git`  
3- `cd pro1oh1`  
4- `docker-compose up` or `docker-compose up -d` for running in the background.
